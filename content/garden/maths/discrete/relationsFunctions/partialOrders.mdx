# Partial & Total Orders

## Partial Order

Hasse Diagram something with anti-symmetric and transitive

The word partial is used to indicate that not every pair of elements needs to be comparable; that is, there may be pairs for which neither element precedes the other.

### Weak Partial Order

is a homogeneous relation < = on a set P that is reflexive, antisymmetric, and transitive.

### Strong Partial Order

is a homogeneous relation < on a set P that is irreflexive, asymmetric, and transitive
